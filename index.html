¿Qué es la Web?

La Web es un sistema de información que permite acceder a documentos y recursos a través de Internet usando un navegador. Funciona mediante tecnologías como HTML, CSS y JavaScript.

Tecnologías principales

  • HTML - Estructura
  • CSS - Diseño
  • JavaScript - Interactividad
